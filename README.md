# Real-Time Document Q&A Application

This project is a simple Streamlit-based web application that allows users to ask questions from text documents in real time.

## Problem Statement
Users often need quick answers from frequently changing documents without manually searching through files.

## Solution
This application continuously monitors documents and provides instant answers based on the latest content.

## Features
- Real-time document reading
- Browser-based user interface
- Automatic updates without restarting

## How to Run
1. Place `.txt` files inside the `documents` folder
2. Run the app using Streamlit
3. Ask questions in the web interface

## Tech Stack
- Python
- Streamlit
- Watchdog
- ngrok

## Use Case
- Office documents
- Notes and policies
- Academic content
